Tool Stack Upgrade: Four Tools That Make Us Faster and Richer

Good news across the board: every one of these plugs directly into Dexter and Kamina so we work with them in-session instead of you doing it by hand. Supabasereal backend · MCP connector
What it is
A real online database with logins, file storage, and security. It is the "back office" that makes an app a real product instead of a demo.
The money
This is what lets us sell real apps. Terri's $22.5k system, Stan's portal, any client app, they all need real data that saves, real logins, and secure PII. Supabase is that. It turns our prototypes into products we can actually charge full price for and hand over.
In practice
Dexter and Kamina can build the database, add logins, and query the data directly while building the app. No more fake demo data, real client records, safely stored.
Connect to Claude Code claude mcp add --transport http supabase https://mcp.supabase.com/mcp First use opens a browser to log into your Supabase and authorize. Same server goes into Kamina's config too.
You do once: make a free Supabase account at supabase.com, then click authorize when the browser opens. That is it. free tier is plenty to start
HiggsfieldAI video + animation · MCP connector
What it is
One doorway to the best AI video and image models (Sora, Kling, Veo, Seedance, Soul, Cinema Studio). Real motion and animation from a prompt or an image.
The money
This is the video upgrade we kept putting off. It IS Leroy's $40 animation tier, the pastlife.era anime motion factory, and real moving content for Stan and any client. Higher-quality videos = more views = more paying clients and higher tiers.
In practice
Dexter or Kamina generates real animated clips in-session and drops them into the video pipeline. Characters and scenes that actually move, not just camera pans.
Connect to Claude Code claude mcp add --transport http higgsfield https://mcp.higgsfield.ai/mcp Hosted server, authorizes through your Higgsfield account in the browser, uses your existing plan credits. No key to manage.
You do once: have a Higgsfield account with credits, then authorize in the browser. Your credits carry over. credit-based, pay for what you generate
Firecrawlweb research + scraping · MCP connector
What it is
Turns any website or web search into clean data the AI can read instantly. Search the web, then pull a page down as tidy text.
The money
This powers our sales research. The lead boards, the growth audits, the pre-call briefs, the deep dives, all of it gets faster and sharper when Dexter can pull real live business data in seconds instead of me piecing it together. More briefs, more outreach, more clients.
In practice
Dexter finds real local businesses, reads their sites, and builds a lead list or an audit live, on his own.
Connect to Claude Code claude mcp add firecrawl -e FIRECRAWL_API_KEY=YOUR_KEY -- npx -y firecrawl-mcp This one uses an API key. You get it, I wire it in (or you paste it into the command).
You do once: sign up at firecrawl.dev and copy your API key. free tier: light daily use, no card needed
Claude Designvisual design · hands off to Claude Code
What it is
Anthropic's own design tool (you already have it on your plan). You design polished web pages, app screens, slides, and one-pagers visually, then it hands the design straight to Claude Code to build for real, no screenshot, no rebuild.
The money
This is our design front door. Mixon Grows websites, client app screens, branded quotes and audits, all designed fast and clean, then built by Dexter. It also learns our black-and-gold brand as a saved design system so everything looks consistent. Faster, better-looking deliverables = premium prices.
In practice
Design the look in Claude Design, import our brand, export to HTML or hand to Claude Code to ship. It reads a codebase and can pull our existing style in.
How it connects Not a plugin, it is a surface on claude.ai. It connects by handing designs to Claude Code natively. Use it at claude.ai, then Dexter picks up exactly where you left off.
You do: nothing to install, it is on your Claude plan already (research preview). Just start using it for the next site or app screen. already available to you
